# jsxgraphbook
Book on how to use [JSXGraph](https://jsxgraph.org)

## Build

* Install `mdbook` from https://github.com/rust-lang/mdBook
* Run `mdbook build` in the root directory of the project
* The output pages are in the subfolder `docs`
